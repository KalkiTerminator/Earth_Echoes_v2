Act as an expert Frontend Developer and UI/UX Designer specializing in React, Three.js, Tailwind CSS, and Framer Motion. 

I want to build an interactive, cinematic 3D web application called "Earth's Echoes". It is an educational visualization of extinct and endangered animal species on a 3D globe.

### Tech Stack
- React (Functional components, Hooks)
- Tailwind CSS (Styling, specific focus on glassmorphism and dark mode)
- Framer Motion (Fluid UI animations, hover/tap states)
- react-globe.gl & Three.js (3D globe rendering)
- Lucide React (Icons)
- Web Audio API (Synthesized UI sound effects)

### Overall Aesthetic & Design System
- **Theme:** Dark, cinematic, premium. Background is a deep neutral-950.
- **UI Elements:** Use Glassmorphism (e.g., `bg-neutral-900/80 backdrop-blur-md border border-neutral-700`).
- **Dynamic Theming:** UI accents and globe atmosphere colors should change based on the selected animal's habitat (e.g., Cyan for Oceans, Emerald for Forests, Slate/White for Tundras).
- **Typography:** Clean, sans-serif, standard modern web fonts.

### Core Features & Layout

1. **The 3D Globe (react-globe.gl)**
   - Display a textured 3D earth (dark map style).
   - The animals should be plotted on the globe using their Latitude/Longitude.
   - **CRITICAL FIX:** Do NOT use WebGL meshes for the animal images, as Wikipedia images will throw CORS/tainted canvas errors. Instead, use HTML overlays (`htmlElementsData` in react-globe.gl).
   - **HTML Pins:** Render each animal as a circular HTML `div` featuring their image (use 'https://wsrv.nl/?url=' as a proxy prefix to bypass image loading errors). Give it a habitat-colored border.
   - **Hover State:** When hovering over an HTML pin, scale it up and reveal a small tooltip with an iconic action (e.g., "✨ Waddling clumsily").
   - Tie the globe's `atmosphereColor` to the selected animal's habitat theme.

2. **Time Machine Timeline (Bottom Center)**
   - A range slider spanning from 10,000 BCE to Present Day.
   - Filtering: Only show extinct animals on the globe if the timeline slider is set to a year *before* or *during* their extinction year.
   - If the slider goes back past 5,000 BCE, visually swap the Globe texture to a darker, more ancient-looking map.

3. **Museum Tour Mode (Top Header)**
   - Include a "Start Museum Tour" button.
   - When clicked, automatically fly the globe's camera (using `globeEl.current.pointOfView`) to each animal one by one, pausing for 10-12 seconds before moving to the next.

4. **Information Panel (Right Sidebar overlay)**
   - When an animal is clicked, slide in a beautifully designed panel from the right.
   - Contents: Animal Name, Scientific Name, Status pill (e.g., "Critically Endangered" in red/orange), a large image, Habitat, Threats, Population/Year Extinct, and a text description.
   - Include a section for an embedded YouTube video iframe.
   - Include a "Listen to Sound" button. (Mock the audio playback if you don't have access to an AI audio generation API, but design the UI for it).

5. **Synthesized UI Sounds (Web Audio API)**
   Please implement a small utility to generate these sounds programmatically so we don't need external audio files:
   - **Hover Sound:** A very subtle, high-pitched "glass tap" (Sine wave at 1200Hz, 0.1s decay).
   - **Click Sound:** A soft, bell-like major third chord (A5 + C#6, 0.2s smooth envelope).
   - **Panel Open Sound:** An elegant, airy "perfect fifth" sweep up (A4 to E5 sliding up an octave, 0.5s fade).
   Attach these to all clickable buttons and interactive globe elements.
